<div align="center">
  <h2>Installation</h2>
</div>

Run this (read the [source code](install.sh) before running!):

        $ curl -fsSL https://astrophena.name/dotfiles/install | bash

<div align="center">
  <h2>Updating</h2>
</div>

Just run `u`. It will pull the latest changes, symlink new files and reload `~/.bashrc`.
