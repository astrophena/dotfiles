# `~/src/dotfiles`

This is [astrophena]'s [dotfiles].

## Install

Run this:

```sh
$ git clone https://github.com/astrophena/dotfiles.git ~/src/dotfiles
$ cd ~/src/dotfiles
$ script/bootstrap
```
